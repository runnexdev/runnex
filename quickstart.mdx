description: "Try Runnex in under 5 minutes"
---

Yes - it's just 2 steps.

1. Install the Runnex GitHub App
   Login to [runnex.dev](https://runnex.dev/login) and follow the prompts to install the Runnex GitHub App.

2. Switching to Runnex Runners in Workflow Files
   Switch your workflow files to use Runnex Runners.
